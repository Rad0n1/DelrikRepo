Installatie details voor P5.JS + node.js voor web server

Download node.js hier: https://nodejs.org/en/download/
Execute de installer
Open cmd (windows + r, cmd, enter)
Typ: npm install -g live-server

Ga met cmd naar de folder waar javascript in staat (De folder genaamd P5 JS)
cd filepath   <<----- Command voor het verplaatsen in mappen
D:  <<----- Command om naar de D:\ schijf te gaan (andere letters werken ook)

Als je in de folder bent met cmd execute deze lijn:
live-server

Dit start de live-server en start hem automatisch in de browser. Als je nu typt en opslaat wordt het automatisch refreshed op de browser.

Vergeet niet je .js file die je aan het aanpassen bent te includen in de html file
